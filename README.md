# GoodBits ToDo App
## Features
1. ToDo List
2. Add ToDo
3. Update ToDo
4. Delete ToDo
5. Delete Modal
6. Toggle ToDo

## Additional Features
1. Deleted ToDo List
2. Deleted ToDo List page visible on small screen
3. Forever Detele
4. Restore Deleted ToDo
